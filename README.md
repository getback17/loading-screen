# Loading Screen

Анимированный экран загрузки на HTML, CSS и JavaScript.

## Описание

Страница показывает анимированный спиннер во время загрузки, после чего плавно появляется основной контент.

## Стек

- **Frontend:** HTML, CSS, JavaScript (Vanilla)

## Как запустить

Открой файл `index.html` в браузере — готово!

## Источник проекта

Проект выполнен на основе туториала из каталога [project-based-learning](https://github.com/practical-tutorials/project-based-learning#html-and-css).
